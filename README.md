# Micro LM

A tiny language-model simulator where users feed a small dataset, watch word associations form, and generate a pixel creature from the learned patterns.

## Version 0.1 Goals

- User enters 5-10 training sentences
- Build a simple word association model
- Visualize learned words and connections
- Generate a creature from dataset statistics
- Allow simple text generation from learned patterns

## Future Goals

- Multiple creature generations
- Ecosystem simulation
- Personality sliders
- Learning progression levels
- Visual AI education system
